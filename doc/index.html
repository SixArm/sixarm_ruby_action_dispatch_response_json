Introduction

This gem extends ActionPack Response with JSON API methods.

You can replace this:

JSON.parse(response.body)

With this:

response.json

You can replace these:

response.json["data"]
response.json["errors"]
response.json["linked"]
response.json["links"]
response.json["meta"]

With these:

response.data
response.errors
response.linked
response.links
response.meta

For docs go to sixarm.com/sixarm_ruby_action_dispatch_response_json/doc

Want to help? We're happy to get pull requests.
